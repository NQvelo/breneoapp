# Styles Directory

Global styles and theme configurations.

## Structure

- `globals.css` - Global CSS styles
- `variables.css` - CSS custom properties/variables
- `theme.ts` - Theme configuration (if using CSS-in-JS)

## Best Practices

- Use Tailwind CSS for utility classes
- Define custom CSS variables for theme colors
- Keep component-specific styles with components
- Use this directory only for truly global styles
